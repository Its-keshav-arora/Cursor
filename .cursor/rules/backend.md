## Scope

These rules apply **only** to files in `TaskFlow-Backend/**`:
- Express routes in `TaskFlow-Backend/src/routes/**`
- Mongoose models in `TaskFlow-Backend/src/models/**`
- Services/middleware in `TaskFlow-Backend/src/services/**`, `TaskFlow-Backend/src/middleware/**`

Do **not** apply these rules to React components or any code in `TaskFlow-API/**`.
